Turtle Project

About the Project

This project is about creating a graphical scene using Python’s Turtle graphics module. I used multiple functions (draw_boat, draw_sail, draw_waves, and draw_bird) in order to create a scene depicting a boat floating in the ocean. I refactored these functions into simplier codes by looping the original code to create shapes such as a rectange, triangle, and half-circles to create the waves. The function "for _ i in range()" is the code responsible for creating the loop.

Sample Code

The first image below creates a boat shape using a looped function to draw a rectangle. The function directs turtle(t) to create the shape of the boat by looping two sides of the rectangle in the function "range(2)". Two of the images below use the looped funtion "range()" with the number inside the parentheses being how many times it will loop. The draw_bird function uses "for angle in ()" instead of "range()" because this function draws the loop of the birds in different positions. The draw_sail function does not use a looped function because it creates a triangle and requires no refactoring. Description of draw_boat looped function:

  1. 1. t.forward(200): Turtle moves forward 200 units
  2. 2. t.left(90): Turtle turns 90° left
  3. t.forward(60): Turtle moves forward 60 units
  4. t.left(90): Turtle turns left again to get ready for next side
